# Monthly Sales Visualization

### Objective

The purpose of this project is to analyze a company’s monthly sales data over a 12-month period. The program calculates important sales metrics, identifies the best and worst-performing months, measures month-over-month growth, and visualizes the data using charts.

### Tools Used

* Python
* Pandas
* Matplotlib

### Features

1. **Load Dataset**
   Reads a CSV file containing monthly sales records.

2. **Dataset Preview**
   Displays the dataset for quick review.

3. **Sales Summary**
   Calculates:

   * Total annual sales
   * Average monthly sales
   * Highest sales
   * Lowest sales

4. **Best and Worst Month Identification**
   Finds the months with the highest and lowest sales.

5. **Month-over-Month Growth Analysis**
   Calculates the percentage increase or decrease in sales compared to the previous month.

6. **Data Visualization**

   * **Line Chart** showing the monthly sales trend
   * **Highlighted Trend Chart** displaying best and worst months
   * **Bar Chart** comparing monthly sales performance
